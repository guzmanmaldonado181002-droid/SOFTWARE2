Playas de Guatemala

Este proyecto es un sitio web sencillo creado con HTML y CSS, cuyo objetivo es mostrar algunas de las playas más representativas de Guatemala de una forma visual, clara y fácil de entender.

La idea principal fue mantener un diseño limpio y simple, evitando menús complicados o secciones innecesarias, para que cualquiera pueda navegarlo sin perderse.

Objetivo del proyecto

Mostrar información básica sobre las playas de Guatemala

Diferenciar los tipos de playas: Pacífico, Naturales y Caribe

Practicar estructura HTML, estilos CSS y organización de archivos

Aplicar buenas prácticas básicas de diseño web

Este proyecto es principalmente educativo, pensado para aprender y reforzar conceptos de desarrollo web frontend.

Estructura del proyecto

El proyecto está organizado de la siguiente forma:

proyecto_playas_guatemala/
│
├── index.html
├── contacto.html
│── playas.html
├── assets/
│   ├── css/
│   │   ├── normalize.css
│   │   └── styles.css
│   │
│   └── img/
│       ├── principal.jpg
│       ├── pacifico.jpg
│       ├── naturales.jpg
│       ├── caribe.jpg
│       └── (otras imágenes de playas)


index.html → Página principal

contacto.html → Página de contacto

playas.html → Pagina de playas 

styles.css → Estilos personalizados del sitio

normalize.css → Normaliza estilos entre navegadores

img/ → Contiene todas las imágenes usadas en el proyecto

Diseño y estilo

Se utilizan las fuentes Playfair Display y Raleway desde Google Fonts

El diseño es responsive (se adapta a distintos tamaños de pantalla)

Las imágenes están ajustadas para mantener una escala uniforme

Se priorizó la legibilidad y la navegación simple

 Cómo usar el proyecto

Descarga o clona el repositorio

Abre la carpeta del proyecto

Haz doble clic en index.html

Listo, el sitio se abrirá en tu navegador

No necesita servidor ni instalación adicional.

 Tecnologías usadas

HTML5

CSS3

Google Fonts

Nada más. Simple y funcional.

 Notas finales

Este proyecto no busca ser un sitio comercial ni profesional, sino una base sólida para aprender, practicar y mejorar habilidades en desarrollo web.
Es totalmente modificable y ampliable si se desea agregar más secciones, animaciones o funcionalidad con JavaScript.
